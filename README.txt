Python Script to Query Cisco ANZ partner events calendar and return results as a dictionary

run using "python cvent_query_anz.py"\n

You can provide one argument to the script that will be used for searching the current months events calendar.\n
You can also type "ALL" as the argument and it will list all events.

e.g. 
Please enter a string to search for
e.g 'python cvent_query_anz.py ALL' to see ALL training for the current month\n
e.g 'python cvent_query_anz.py Sydney' to see Sydney training for the current month\n
e.g 'python cvent_query_anz.py Webinar' to see Webinar based training for the current month\n
e.g 'python cvent_query_anz.py Live' to see Live instructor led training for the current month\n
